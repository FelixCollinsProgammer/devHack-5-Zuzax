Цели и задачи программы "Камень, Ножницы, Бумага"

Главная цель: Создать интерактивную игру "Камень, Ножницы, Бумага" против компьютера, управляемую жестами рук, распознаваемыми веб-камерой.

Задачи программы:

Распознавание жестов рук:

- Обеспечить распознавание основных жестов "Камень", "Ножницы" и "Бумага" с помощью анализа изображения с веб-камеры.
- Достичь адекватной точности и скорости распознавания в реальном времени.
- Обеспечить визуальную обратную связь для пользователя, отображая распознанные жесты на экране.

Реализация игровой логики:

- Разработать правила игры "Камень, Ножницы, Бумага" для определения победителя каждого раунда на основе выбора пользователя (жеста) и случайного выбора компьютера.
- Реализовать счетчик раундов и общий счет игры (счетчик побед игрока и компьютера).
- Обеспечить игровой цикл, включающий подготовку к раунду, распознавание жеста пользователя, выбор компьютера, определение результата и переход к следующему раунду или отображение итогов.
- Создание пользовательского интерфейса (GUI):
- Разработать интуитивно понятный и визуально привлекательный графический интерфейс пользователя с использованием библиотеки PyQt5.

Реализовать:

- Стартовый экран: для ввода имени игрока и начала игры.
- Игровой экран: для отображения видеопотока с веб-камеры, распознанных жестов, выбора компьютера, результатов текущего раунда, счета и номера раунда.
- Экран результатов: для отображения итоговых результатов всех раундов и определения победителя игры.
- Обеспечить анимации и стилизацию интерфейса для улучшения пользовательского опыта.

Обеспечение стабильной и плавной работы:

- Использовать многопоточность для разделения задач обработки видео и обновления GUI, чтобы избежать зависаний и обеспечить плавность работы программы.
- Обработать возможные ошибки, такие как отсутствие доступа к камере, и корректно информировать пользователя.

Предоставление пользовательского опыта:

- Сделать игру увлекательной и интересной для пользователя.
- Обеспечить обратную связь на действия пользователя, такую как отображение распознанных жестов, результатов раундов и общего счета.
- Предусмотреть возможность перезапуска игры.

Используемые технологии для создания:

- Язык программирования: Python - выбран за свою простоту, читаемость, доступность библиотек для - GUI и компьютерного зрения, а также скорость разработки.
- GUI Framework: PyQt5 - для создания графического пользовательского интерфейса. Предоставляет инструменты для создания окон, кнопок, лейблов, макетов и стилизации интерфейса. Поддержка анимации и многопоточности.
- Библиотека компьютерного зрения: OpenCV (cv2) - для работы с веб-камерой и обработки видеопотока. Используется для захвата кадров, преобразования цветового пространства и отображения видео в GUI.
- Библиотека распознавания жестов рук: MediaPipe (mp.solutions.hands) - разработанная Google для высокопроизводительного отслеживания рук и распознавания жестов. Обеспечивает обнаружение ключевых точек на руках (landmarks) и их соединений, что позволяет реализовать логику распознавания жестов.
- Математическая библиотека: math - для математических вычислений, в частности для расчета расстояний между ключевыми точками рук при распознавании жестов.
Библиотека случайных чисел: random - для генерации случайного выбора компьютера ("Камень", "Ножницы", "Бумага").
- Таймеры (QTimer из PyQt5): - для реализации обратного отсчета перед началом раунда и для создания задержки между раундами для отображения результатов.
- Многопоточность (QThread из PyQt5): - для создания отдельного потока для обработки видео с веб-камеры, чтобы GUI оставался отзывчивым и не зависал во время обработки кадров.

Краткая инструкция к игре "Камень, Ножницы, Бумага":

1. Запустите игру.
2. На стартовом экране: введите никнейм и нажмите "Играть!".
3. В игре: подождите обратный отсчет (3, 2, 1), затем покажите жест "Камень", "Ножницы" или "Бумага" в камеру.
4. Смотрите результат раунда. Повторите для 5 раундов.
5. На экране "Результаты": узнайте итог игры. Нажмите "Переиграть" для новой игры.
